FCC Full Stack JavaScript Curriculum Agreement

This table outlines the curriculum for the FCC Full Stack JavaScript program and the time I plan to spend on each module.

Curriculum Overview/Timeline
Module Date Completed Description
Code Editors (5 steps) Oct 6-7 Learn how to use different code editors and set up your development environment efficiently.
Variables and Strings (62 steps) Oct 6-7 Understand how to declare variables and manipulate strings using JavaScript syntax and operations.
Booleans and Numbers (37 steps) Oct 8-9 Explore numeric and boolean data types, arithmetic operations, and logical comparisons.
Functions (38 steps) Oct 8-9 Learn how to write reusable blocks of code using functions, parameters, and return values.
Arrays (35 steps) Oct 10 Work with arrays to store, access, and manipulate lists of data efficiently.
Objects (28 steps) Oct 13-14 Understand how to use objects to represent structured data with key-value pairs.
Loops (22 steps) Oct 13-14 Master for, while, and do-while loops to perform repetitive tasks in your code.
JavaScript Fundamentals Review (30 steps) Oct 15 Reinforce the foundational concepts of JavaScript learned in previous modules.
Higher Order Functions and Callbacks (36 steps) Oct 15 Learn how to use functions that take other functions as arguments or return them as results.
DOM Manipulation and Events (87 steps) Oct 16 Interact with web pages by selecting, updating, and responding to user events in the DOM.
JavaScript and Accessibility (42 steps) Oct 16 Ensure your web content is accessible to all users by implementing accessible JavaScript practices.
Debugging (8 steps) Oct 16-17 Learn debugging tools and techniques to identify and fix errors in your JavaScript code.
Basic Regex (47 steps) Oct 17 Use Regular Expressions to search, match, and manipulate patterns in text.
Form Validation (103 steps) Oct 20-21 Validate user input in forms to ensure data accuracy and improve user experience.
Dates (5 steps) Oct 20-21 Work with the Date object to manage and format dates and times in JavaScript.
Audio and Video Events (55 steps) Oct 22-23 Control multimedia elements and respond to playback events programmatically.
Maps and Sets (37 steps) Oct 22-23 Learn how to use Map and Set data structures for efficient key-value storage and unique collections.
localStorage and CRUD Operations (78 steps) Oct 24-27 Store, retrieve, and modify data locally in the browser using CRUD operations.
Classes (68 steps) Oct 24-27 Use classes to create reusable, object-oriented components and manage inheritance.
Recursion (112 steps) Oct 28 Understand recursive problem solving and how functions can call themselves to break down complex tasks.
Functional Programming (23 steps) Oct 30-31 Learn pure functions, immutability, and higher-order techniques for cleaner, modular code.
Asynchronous JavaScript (36 steps) Oct 30-31 Handle asynchronous tasks using callbacks, promises, and async/await for smoother program flow.
